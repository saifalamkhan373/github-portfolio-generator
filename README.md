# Portfolio Generator by Saif Alam Khan

### Requirements

- Node.js
- npm or yarn

### Getting Started

Clone this repository using

```bash
git clone https://github.com/saifalamkhan373/github-portfolio-generator.git
```

Install dependencies with

```bash
npm install #or
yarn
```

And then run the development server

```bash
npm start #or
yarn start
```
